# QA Proyecto Integrador

## Descripción

Este repositorio fue creado como parte de una actividad integradora del taller de Testing QA.  
Simula el trabajo de un tester dentro de un proyecto real, aplicando buenas prácticas de documentación, organización de casos de prueba y uso de control de versiones con Git y GitHub.

---

## Contenido del repositorio

El repositorio contiene:

- Casos de prueba manuales  
- Documentación básica del proceso de testing  
- Evidencias de pruebas realizadas  

### Estructura

qa-proyecto-integrador/
│
├── README.md
├── casos-prueba.txt
└── evidencias/
└── nota.txt

## Objetivo del proyecto

- Practicar el diseño de casos de prueba manuales  
- Aplicar organización y trazabilidad en testing  
- Simular el flujo de trabajo de un QA  
- Utilizar Git para el control de versiones  
- Publicar documentación en GitHub  

---

## Alcance del testing

Los casos de prueba incluidos contemplan validaciones sobre funcionalidades comunes de sistemas web como:

- Login
- Registro de usuarios
- Validaciones de campos
- Manejo de sesión
- Seguridad básica
